# Educational LMS Platform (MVP)

Система для розміщення та дистрибуції інформаційних навчальних матеріалів.

## Основні сутності (Entities)
* **Student (Учень):** Користувач системи, який отримує доступ до матеріалів.
* **Cohort (Група):** Сутність для об'єднання учнів (наприклад, міні-групи по 4 особи).
* **Material (Матеріал):** Контент (переважно PDF), доступ до якого регулюється на основі приналежності до групи та часових обмежень (access_until).

## Стек технологій
* Python
* Django (MVT architecture)
* SQLite (Development) / PostgreSQL (Production)
